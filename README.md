# React_code_example
## Состояния React.  
Служат для изменения компонентов после парсинга 
страницы (отработки всех функций). Нужны для динамических изменений на какие либо события.
Работа с состояниями реализуется при помощи функции (она же хук) библиотеки
React `{ useState }`. Хук легко определить по тому что он начинается со 
слова `use`. 
Особенности работы:
1. Делаем именованный импорт в компонент где будут изменения `import React, { useState } from "react";`
2. Вызываем внутри компонента `useState()` ни в коем случае не внутри вложенных элементов компонента.
3. Передаем внутр `useState()` значение по умолчанию переменной которое в дальнейшем подлежить изменению.
`useState(props.description)`  
`useState` возвращает массив из двух элементов 1-значение переменной, 2-функция для обновления состояния.
4. Вызываем функцию для обновления состояния и передаем в нее нужное нам значение `setDescription ('New cost');`
при этом происходит присваивание значения переменной которую возвращает `useState` из массива + 
перезапуск компонента в котором работает `useState`
